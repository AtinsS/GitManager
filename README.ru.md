# 🚀 Git Менеджер Репозиториев от AtinsS

  

**Версия:** 1.1.1

**Платформа:** Windows  

**Язык:** Batch Script

  

Удобный консольный менеджер для тех кто устал вводить команды Git вручную.

Позволяет легко управлять несколькими репозиториями через простое меню. ТОЛЬКО ДЛЯ WINDOWS!

  

## 📋 Возможности

  

### Основные функции:

- 📁 Управление списком репозиториев 
	 - Добавление, Удаление, распределение по группам.

- 🔄 Массовое обновление всех репозиториев

- 🌐 Клонирование новых репозиториев

- 📊 Просмотр статуса Git

- ⬇️ Git pull (обновление)

- ⬆️ Git push (отправка)

- 📝 Commit с комментарием

- ⚡ Быстрый commit + push с авто-комментарием

- 📜 Просмотр истории (git log)

- 🌿 Создание и переключение веток

- 🤖 Автоматические коммиты по расписанию

  

### Дополнительно:

- 💾 Сохранение списка репозиториев в конфиг файл

- 🔍 Проверка наличия Git

- 🛡️ Обработка ошибок и предупреждений

- 👅 Двуязычная система определяет язык автоматически при запуске


  

## 📦 Требования

  

- Windows 10/11 (с поддержкой ANSI цветов)

- Git for Windows ([скачать](https://git-scm.com/download/win))

- Windows Terminal (рекомендуется) или любой современный терминал

  

## 🔧 Установка и использование

  

1. **Нужен Git**

2.  Скачайте архив и запустите

```batch

GIT-MANAGER.BAT

```

### Главное меню:

```

  🚀 GIT МЕНЕДЖЕР - Управление репозиториями
    ═══════════════════ By AtinsS ═══════════════════

    Список репозиториев:

    📁 Группа: JS
    1.  JS-Learn
    2.  JS-FINAL
    3.  Crop-IMG

    📁 Группа: OTHER
    4.  GIT-MANAGER

    📁 Группа: My-Vault
    5.  Vault

    Меню действий:
    1-5: Выбрать репозиторий по номеру
    C: Клонировать новый репозиторий
    A: Добавить существующий локальный репозиторий
    U: Обновить все репозитории
    G: Управление группами
    D: Удалить репозиторий из списка
    S: Настройки
    X: Выход

    ⚡ Ваш выбор:

```

  

### Меню репозитория:

```

 ════════════════════════════════════════════════════════════
    Репозиторий: JS-Learn    Ветка: dev
```

## ⚠️ Возможные проблемы и решения

  

### Не работают цвета:

- Используйте Windows Terminal вместо стандартной консоли

- Или включите поддержку ANSI в реестре

  

### Git не найден:

- Установите Git for Windows

- Добавьте Git в переменную PATH

  

### Ошибка "Папка не найдена":

- Проверьте путь в конфиге

- Используйте опцию "Указать новый путь"
