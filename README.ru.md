# 🚀 Git Manager

> Git без боли. Без команд. Без лишних действий.

**Платформа:** Windows  
**Язык:** Batch Script

---

## 🚀 Что это?

**Git Manager** — это интерактивная оболочка над Git, которая превращает сложный CLI в понятное меню.

Забудь про это:

```
git add .
git commit -m "fix"
git push origin main
```

Теперь — просто выбери действие.  
**Остальное сделает инструмент.**

---

## ⚡ Почему это удобно

- ⚡ **Быстрее** — рутинные операции в пару нажатий
    
- 🧠 **Проще** — не нужно помнить команды
    
- 📦 **Удобнее** — все репозитории в одном месте
    
- 🎯 **Практичнее** — меньше ошибок, больше фокуса на коде
    

---

## 🧰 Фичи проекта

- 📁 Управление несколькими репозиториями
    
- 📦 Распределение проектов по группам
    
- ⬆️ Commit + Push за одно действие
    
- ⬇️ Массовое обновление всех репозиториев одной кнопкой
- 📊 Просмотр статуса репозитория сразу из главного меню
    
- 🌿 Полноценная работа с ветками
    
- 🔄 Clone / Pull / Push без CLI
    
- 📜 Просмотр истории (git log)
    
- 🤖 Автоматические коммиты по таймеру
- 🌐  Автоматический подбор языка(пока только 2, Русский и Английский)

## ⚙️ Быстрый старт

1. Установи [Git](https://git-scm.com/)
    
2. Скачай проект, либо склонируй репозиторий
    
3. Запусти: `GIT-MANAGER.BAT`

4. Кайфуй 
---

## 🖥️ Как это выглядит

### Главное меню

```batch
════════════════════════════════════════════════════════════
                        GIT MANAGER  🚀
                         by AtinsS
════════════════════════════════════════════════════════════

▸ Нет репозиториев

════════════════════════════════════════════════════════════
▸ ДЕЙСТВИЯ
    [C] Клонировать   [A] Добавить    [U] Обновить всё
    [G] Группы        [D] Удалить     [S] Настройки
    [X] Выход
════════════════════════════════════════════════════════════

  →

```

---

### Работа с репозиториями

```batch
▸ Группа: Group1
  1. Repo1  [dev] ● чистый

▸ Группа: Group2
  2. Repo2  [main] ● изменения
  3. Repo3  [main] ● чистый
```

---

### Действия внутри репозитория

```batch
Репозиторий: Repo1  Ветка: dev
Статус: ✅ чисто

1. Git status
2. Git pull
3. Commit + Push
4. Отмена изменений
5. История (log)
6. Merge
7. Отмена merge
8. Ветки для merge
9. Создать ветку
10. Переключить ветку
11. Авто-коммиты
```
